Important FAQ

What is the Document Object Model

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

How do you select an element from the DOM?

To select id or Class by query selector: 1.document.querySelector(#id/.class) 2.document.getElementById('id name') 3.document.getElementByClass('id name')

What is event delegation in the contect of DOM,and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an element's attributes and styles using the DOM?

To perform any form of DOM manipulation, you have to access the document object first. Next we have the html root element which is a child of the document object. Next on the line are the body and head elements which are siblings to each other and children of the html element.
